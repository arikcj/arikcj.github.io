# arikcj.github.io

This is the repository for CHEN Jie (陈杰)'s personal academic homepage, built with Jekyll and deployed on GitHub Pages.

## Content Description

- `index.html`: Personal homepage, including introduction, contact information, research interests, education background, work experience, awards and news.
- `publications/`: Publications and papers list page.
- `activities/`: Academic activities page.
- `teaching/`: Teaching and courses page.
- `_layouts/`: Site layout templates.
- `_config.yml`: Jekyll configuration file.
- `css/main.css`: Global stylesheet.
- `static/`: Site static resources, such as images.

## Local Preview

If you want to preview the site locally, install Jekyll and run:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://127.0.0.1:4000`.

## Notes

This repository is mainly used to display personal homepage content and is automatically deployed via GitHub Pages.
