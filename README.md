# arikcj.github.io

这是 CHEN Jie (陈杰) 的个人学术主页仓库，基于 Jekyll 构建，并部署在 GitHub Pages 上。

## 内容说明

- `index.html`：个人主页，包含简介、联系信息、研究方向、教育背景、工作经历、奖励与新闻。
- `publications/`：论文与出版物列表页面。
- `activities/`：学术活动页面。
- `teaching/`：教学与课程页面。
- `_layouts/`：站点布局模板。
- `_config.yml`：Jekyll 配置文件。
- `css/main.css`：全局样式文件。
- `static/`：站点静态资源，如图片。

## 本地预览

如果你想在本地预览网站，可以安装 Jekyll，然后运行：

```bash
bundle install
bundle exec jekyll serve
```

之后访问 `http://127.0.0.1:4000`。
